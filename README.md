# Retail-sales-data-report-

## Overview
This project presents an interactive Retail Sales Performance Dashboard built in Microsoft Excel. The dashboard provides a clear view of revenue, profit, sales trends, product performance, regional performance, and profitability insights.

## Dashboard Features
- Total Revenue, Quantity Sold, Total Profit, and Profit Margin KPIs
- Monthly sales trend analysis
- Regional sales performance breakdown
- Product revenue and profit comparison
- Quantity analysis by product
- Profitability insights using scatter plot analysis
- Interactive slicers for product and region filtering

## Tools Used
- Microsoft Excel
- PivotTables
- PivotCharts
- Slicers
- Data Cleaning
- Dashboard Design

## Key Insights
- Sales peaked in Q3 (September), indicating strong seasonal demand.
- The West region generated the highest sales, making it the top-performing market.
- Laptops and headphones generated the highest revenue and profit.
- Monitors recorded the lowest profitability despite moderate sales.

## Recommendations
- Increase focus on top-performing products such as laptops and headphones to maximize revenue and profit.
- Review pricing, sourcing, or operational costs for monitors to improve profitability.
- Replicate successful sales strategies from the West region in lower-performing regions.
- Prepare inventory and marketing campaigns ahead of Q3 to capitalize on seasonal demand.
- Use targeted promotions to improve performance of underperforming products.
